## 1.1 Python 后端（15 个示例）

### 示例 1：FastAPI 微服务
**原始:** 使用 Python 开发后端接口，基于 Flask 框架实现业务逻辑，使用 MySQL 存储数据。
**优化:**
技能：RESTful API 开发、微服务架构、异步编程、数据库设计、容器化部署
项目简介：基于 FastAPI+Uvicorn 构建高性能异步 API 服务，实现 RESTful 接口与自动文档，使用 MySQL 存储业务数据，Redis 缓存热点数据，Pytest 编写单元测试，Docker 容器化部署，支撑日均 100 万次 API 调用，接口 P99 延迟<100ms。

### 示例 2：Django 全栈
**原始:** 负责 Django 项目开发，实现用户管理、权限控制等功能。
**优化:**
技能：Django 全栈开发、RBAC 权限管理、异步任务调度、前后端分离
项目简介：基于 Django+DRF 开发企业级应用，实现用户认证、RBAC 权限控制、数据管理等功能，PostgreSQL 存储业务数据，Celery+Redis 异步任务调度，Vue.js 开发管理后台，运营效率提升 40%。

### 示例 3：高并发接口
**原始:** 开发高并发接口，使用 Python 和 Redis，优化接口性能。
**优化:**
技能：高并发后端开发、接口性能优化、消息队列、多级缓存
项目简介：基于 FastAPI 开发高并发 API，设计 Redis+ 本地多级缓存，RabbitMQ 异步解耦与流量削峰，连接池优化与 SQL 调优，接口 QPS 从 500 提升至 3000，P99 延迟降低至 80ms。

### 示例 4：数据爬取
**原始:** 使用 Python 爬虫抓取数据。
