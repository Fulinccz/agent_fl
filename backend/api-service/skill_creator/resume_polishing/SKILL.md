# 简历内容润色技能 (ResumePolishingSkill)

## 作用
将大白话转换为专业简历话术，提升简历的专业性和吸引力。

## 输入
- **原始经历**：用户的原始工作/项目描述
- **JD/岗位**（可选）：目标岗位描述，用于针对性优化

## 执行流程

### 1. RAG 检索行业动词库
从知识库检索与用户经历相关的：
- 行业专业动词（如：架构、优化、重构、赋能、驱动）
- 技术术语和标准表达
- 岗位关键词

### 2. 句式优化
**优化原则：**
- 使用 STAR 法则（Situation, Task, Action, Result）
- 强调动作和成果
- 量化指标优先

**转换示例：**
```
原始：做了个后端接口，用 Flask 写的
优化：基于 Flask 框架设计并实现 RESTful API 后端服务

原始：负责带新人，教他们怎么写代码
优化：主导团队新人培养，建立代码规范培训体系，提升团队整体开发效率 30%

原始：修了很多 bug
优化：主导系统稳定性优化，累计修复核心 bug 50+，系统故障率降低 80%
```

### 3. 去口语化
**替换规则：**
- "做了" → "设计并实现"、"主导开发"
- "负责" → "主导"、"牵头"、"统筹"
- "帮忙" → "协助"、"支持"、"协同"
- "写了" → "开发"、"构建"、"实现"
- "搞过" → "主导"、"负责"
- "弄了" → "设计"、"搭建"

### 4. 专业术语增强
根据 JD 关键词，自动融入：
- 技术栈名称（如：FastAPI、Kubernetes、TensorFlow）
- 方法论（如：敏捷开发、CI/CD、DevOps）
- 行业术语（如：高并发、微服务、分布式）

## 输出
优化后的专业描述，包含：
- ✅ 专业动词开头
- ✅ 量化成果
- ✅ 技术关键词
- ✅ STAR 结构

## 使用示例

### 输入
```
原始经历：
- 做了个 Python 后端，用 Flask 写的，能处理用户注册登录
- 帮忙优化了数据库查询，快了很多
- 带了 2 个新人，教他们写代码

JD：Python 后端开发工程师
```

### 输出
```
优化后：
- 基于 Flask 框架设计并实现用户认证系统，支持注册、登录、权限管理等核心功能，日均处理请求 10 万+
- 主导数据库查询性能优化，通过索引优化和查询重构，接口响应时间从 500ms 降至 50ms，提升 90%
- 担任技术导师，培养 2 名初级工程师，建立代码审查机制，团队代码质量提升 40%
```

## 实现要点

1. **动词库检索**：使用 RAG 从知识库检索相关行业动词
2. **句式模板**：预定义 STAR 句式模板，自动套用
3. **口语检测**：识别并替换口语化表达
4. **关键词匹配**：根据 JD 自动融入技术关键词
5. **量化建议**：提示用户补充可量化的指标

## 注意事项

- 保持真实性，不夸大或虚构
- 尊重用户原意，不过度修饰
- 量化指标需用户确认
- 技术关键词需与用户经历匹配
