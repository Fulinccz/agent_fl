# 简历整体评分技能 (ResumeScoreSkill)

## 作用
对简历进行多维度综合评分，提供量化评估和改进建议，帮助用户了解简历质量和优化方向。

## 输入
- **完整简历**：用户的完整简历内容
- **JD**（可选）：目标岗位描述，用于匹配度评分
- **岗位类型**（可选）：帮助调整评分权重

## 评分维度

### 1. JD 匹配度（40% 权重）
**评估简历与岗位的匹配程度**

#### 评分标准（0-100 分）
- **90-100 分**：完美匹配
  - 核心技能 100% 覆盖
  - 关键经验高度相关
  - 任职要求全部满足
  
- **80-89 分**：高度匹配
  - 核心技能 80%+ 覆盖
  - 主要经验相关
  - 关键要求满足
  
- **70-79 分**：基本匹配
  - 核心技能 60-80% 覆盖
  - 部分经验相关
  - 基本要求满足
  
- **60-69 分**：勉强匹配
  - 核心技能 50-60% 覆盖
  - 经验相关性一般
  - 部分要求缺失
  
- **<60 分**：不匹配
  - 核心技能覆盖<50%
  - 经验不相关
  - 关键要求缺失

#### 评估要素
1. **关键词匹配**（30%）
   - 技术栈重合度
   - 职责描述匹配度
   - 能力要求覆盖度

2. **经验匹配**（40%）
   - 工作年限符合度
   - 行业经验相关性
   - 项目经验匹配度

3. **技能匹配**（30%）
   - 核心技能掌握度
   - 技能深度与岗位要求匹配
   - 技能广度覆盖

### 2. 专业度（25% 权重）
**评估简历的专业性和规范性**

#### 评分标准（0-100 分）
- **90-100 分**：非常专业
  - 使用行业标准术语
  - 句式专业规范
  - 无口语化表达
  - 格式统一美观
  
- **80-89 分**：专业
  - 大部分术语准确
  - 句式较为规范
  - 少量口语化
  - 格式基本统一
  
- **70-79 分**：较为专业
  - 术语使用基本正确
  - 句式一般
  - 有一定口语化
  - 格式基本规范
  
- **60-69 分**：一般
  - 术语使用不准确
  - 句式随意
  - 口语化明显
  - 格式不够统一
  
- **<60 分**：不专业
  - 术语错误
  - 句式混乱
  - 大量口语化
  - 格式混乱

#### 评估要素
1. **用词专业性**（40%）
   - 行业术语使用
   - 动词专业性
   - 技术名词准确性

2. **表达规范性**（30%）
   - 句式结构
   - 时态一致性
   - 逻辑清晰度

3. **格式规范性**（30%）
   - 排版美观度
   - 字体字号统一
   - 间距合理性

### 3. 数据化程度（20% 权重）
**评估简历中量化指标的使用情况**

#### 评分标准（0-100 分）
- **90-100 分**：高度数据化
  - 80%+ 经历有量化指标
  - 指标具体且有说服力
  - 数据维度丰富（规模、性能、效率等）
  
- **80-89 分**：数据化良好
  - 60-80% 经历有量化指标
  - 指标较为具体
  - 数据维度较丰富
  
- **70-79 分**：数据化一般
  - 40-60% 经历有量化指标
  - 部分指标具体
  - 数据维度一般
  
- **60-69 分**：数据化不足
  - 20-40% 经历有量化指标
  - 指标不够具体
  - 数据维度单一
  
- **<60 分**：缺乏数据化
  - <20% 经历有量化指标
  - 指标模糊
  - 几乎没有数据

#### 评估要素
1. **量化覆盖率**（50%）
   - 有数据的经历占比
   - 关键成就是否量化

2. **数据质量**（30%）
   - 数据具体性（精确数字 vs 模糊描述）
   - 数据可信度
   - 数据维度多样性

3. **数据呈现**（20%）
   - 数据位置醒目度
   - 数据对比（提升 X%、降低 Y%）
   - 数据与成果的关联

### 4. 结构完整性（10% 权重）
**评估简历结构的完整性和逻辑性**

#### 评分标准（0-100 分）
- **90-100 分**：结构完美
  - 必备模块齐全（基本信息、教育、经历、技能）
  - 模块顺序合理
  - 逻辑清晰
  - 重点突出
  
- **80-89 分**：结构完整
  - 必备模块齐全
  - 顺序基本合理
  - 逻辑较清晰
  
- **70-79 分**：结构基本完整
  - 主要模块具备
  - 顺序一般
  - 逻辑基本清晰
  
- **60-69 分**：结构不完整
  - 缺少部分模块
  - 顺序不合理
  - 逻辑不够清晰
  
- **<60 分**：结构混乱
  - 缺少关键模块
  - 顺序混乱
  - 逻辑不清

#### 评估要素
1. **模块完整性**（50%）
   - 基本信息
   - 教育背景
   - 工作/项目经历
   - 专业技能
   - 其他加分项（证书、奖项等）

2. **逻辑清晰度**（30%）
   - 时间顺序
   - 重要性排序
   - 模块间逻辑

3. **重点突出**（20%）
   - 核心优势展示
   - 关键经历突出
   - 亮点明确

### 5. 亮点突出度（5% 权重）
**评估简历是否有记忆点和差异化优势**

#### 评分标准（0-100 分）
- **90-100 分**：亮点非常突出
  - 有明确的差异化优势
  - 核心成就令人印象深刻
  - 个人品牌清晰
  
- **80-89 分**：亮点突出
  - 有明显优势
  - 成就较为突出
  - 个人特色明显
  
- **70-79 分**：有亮点
  - 有一定优势
  - 部分成就突出
  - 个人特色一般
  
- **60-69 分**：亮点不明显
  - 优势不明显
  - 成就一般
  - 缺乏个人特色
  
- **<60 分**：无亮点
  - 无明显优势
  - 成就平庸
  - 同质化严重

#### 评估要素
1. **差异化优势**（50%）
   - 独特技能组合
   - 特殊项目经验
   - 行业影响力

2. **成就突出度**（30%）
   - 核心成就影响力
   - 成果可衡量性
   - 成就稀缺性

3. **个人品牌**（20%）
   - 个人特色标签
   - 专业形象塑造
   - 记忆点设计

## 实现要点

1. **多维度评分**：5 个维度独立评分后加权
2. **RAG 辅助**：使用 RAG 检索行业标准和对标简历
3. **关键词分析**：自动提取和匹配 JD 关键词
4. **数据检测**：识别简历中的量化指标
5. **口语检测**：识别并标记口语化表达
6. **智能建议**：基于问题生成具体改进建议

## 注意事项

- ✅ 评分客观公正，有明确依据
- ✅ 建议具体可执行，不空洞
- ✅ 指出问题的同时给出改进方案
- ✅ 鼓励为主，建设性批评
- ✅ 考虑不同行业的特点和标准

## 扩展功能

### 1. 行业基准对比
- 与同行业同岗位简历对比
- 提供行业平均分和优秀线
- 显示竞争力百分位

### 2. ATS 模拟评分
- 模拟主流 ATS 系统评分
- 提供关键词优化建议
- 预估初筛通过率

### 3. 面试官视角评估
- 从面试官角度评估简历
- 预测可能的面试问题
- 指出可能的质疑点

### 4. 追踪改进效果
- 保存历史评分记录
- 对比优化前后变化
- 追踪求职进展
